DIGITALIZATION OF A SOLID OBJECT USING KINECT AND 3D PRINTING

Esta investigación propone una metodología innovadora para digitalizar objetos sólidos mediante el escaneo 3D con Kinect. Los fotogramas obtenidos se procesan en KScan 3D para crear una imagen tridimensional. Luego, en MeshMixer se rellenan las partes huecas y se genera un archivo STL. Con Blender, se ajustan las coordenadas y se imprimen en 3D las figuras reconstruidas.
